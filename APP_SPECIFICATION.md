# UMOJA FEMSPACE APP SPECIFICATION

## Slogan
Start • Scale • Succeed

## App Purpose
A client-friendly microfinance platform enabling customers to apply for loans, track repayments, manage guarantors, and access financial services digitally.

---

# MODULE 1: USER ACCOUNTS

Features:
- Registration
- Login
- Phone Verification
- Profile Management
- ID Verification (KYC)
- Next of Kin
- Security Settings

---

# MODULE 2: LOANS

Loan Products:
- Business Loan
- Emergency Loan
- Women Enterprise Loan
- Youth Loan
- Agriculture Loan
- School Fees Loan
- Asset Financing Loan
- Group Loan

Features:
- Loan Application
- Loan Calculator
- Loan Status Tracking
- Loan Statements
- Loan History
- Loan Top-Up Requests
- Loan Renewals
