# UMOJA FEMSPACE USER ROLES

## 1. CLIENT

Permissions:
- Register account
- Update profile
- Apply for loans
- View loan status
- View repayment schedule
- Upload documents
- View statements
- Contact support

---

## 2. LOAN OFFICER

Permissions:
- View applications
- Verify client details
- Verify guarantors
- Recommend approval
- Recommend rejection
- Monitor repayments
- Generate reports

---

## 3. CREDIT MANAGER

Permissions:
- Review applications
- Approve loans
- Reject loans
- Set loan limits
- Manage risk assessments

---

## 4. COLLECTION OFFICER

Permissions:
- View overdue loans
- Record collection notes
- Monitor repayments
- Manage recovery actions

---

## 5. SYSTEM ADMINISTRATOR

Permissions:
- Manage users
- Manage staff accounts
- Configure loan products
- Generate system reports
- Manage notifications
- View audit logs

---

## 6. SUPER ADMIN

Permissions:
- Full system access
- Manage all users
- Manage all loans
- View all reports
- Configure system settings
- Assign staff roles
